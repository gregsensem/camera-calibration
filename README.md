#Camera Calibration
This is a script written for finding the camera intrinsic parameters.

### Dependencies:
* OpenCV 3.4
* Cmake 3.5

### Before build:
please change the video source path in calibration.cpp

### To build:
1. git clone
2. cd build
3. cmake ..
4. make

### To Run:
run ./calibration
